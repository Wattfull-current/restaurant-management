# 🍽️ Restaurant Management System

A Java console application that brings order to restaurant operations — 
from browsing the menu to printing the final bill.

## ✨ Features
- 📋 **Menu display** — browse available items with prices
- 🧾 **Order management** — take and track customer orders
- 💰 **Automatic billing** — calculates totals so no one's doing math on a napkin

## 🚀 How to run

javac Restaurantmanagement.java

java Restaurantmanagement


## 🎯 About
Built as a school project to put core Java concepts to work — object-oriented 
programming, control structures, and console-based interaction — in a 
practical, real-world style application rather than just theory exercises.
